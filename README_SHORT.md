# Media Monitoring SaaS Demo

Enterprise-grade Media Monitoring and Social Intelligence platform demo

## Features
- Boolean Query Builder + Real-time Alerting
- Topic Modeling & Trend Detection  
- LLM-Powered Auto Brief Generator
- Crisis Detection & Response System
- Competitive Intelligence Dashboard
- AI-Powered Content Recommendations
- Influencer & Brand Ambassador Tracking

## Tech Stack
- React 18 + TypeScript
- Vite + TailwindCSS  
- Recharts + Lucide React
- Enterprise UI/UX

## Quick Start
```bash
npm install
npm run dev
```
